在写主题样式的时候经常会碰到用背景图铺满整个背景的需求，这里分享下使用方法

需要的效果

  1. 图片以背景的形式铺满整个屏幕，不留空白区域
  2. 保持图像的纵横比（图片不变形）
  3. 图片居中
  4. 不出现滚动条
  5. 多浏览器支持
